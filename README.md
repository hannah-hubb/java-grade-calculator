# Student Grade Calculator (Java)

## Description
This project is a simple Java application developed as part of a beginner-level programming course. It demonstrates basic programming concepts such as user input, variables, conditional statements, and arithmetic operations.

## Purpose
The purpose of this program is to help calculate a student's average marks and assign a corresponding grade. It is designed to simulate a basic grading system used in schools.

## Functionality
- The program asks the user to enter the student's name.
- It then takes marks for three subjects as input.
- It calculates the average of the three marks.
- Based on the average, it assigns a grade using predefined conditions:
  - A (90 and above)
  - B (75–89)
  - C (60–74)
  - D (50–59)
  - F (below 50)
- Finally, it displays the student's name, average marks, and grade.

## Concepts Used
- Variables and Data Types
- User Input using Scanner
- Arithmetic Calculations
- Conditional Statements (if-else)

## How to Run
1. Compile the program using a Java compiler:
   javac Main.java
2. Run the program:
   java Main
